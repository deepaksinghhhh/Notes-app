# 📝 Notes App

A clean and simple notes application built with HTML, CSS, and JavaScript.

## Features
- Add notes with title and description
- Delete notes
- Clean card-based layout
- Responsive design

## Tech Stack
- HTML5
- CSS3 (Grid layout, Card design)
- JavaScript (DOM manipulation)

## How to Run
1. Clone this repository
2. Open index.html in any browser
3. Start adding notes!
